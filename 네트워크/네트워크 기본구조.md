# 네트워크 기본구조

**네트워크?**

여러대의 컴퓨터 시스템이나 정보 기기를 전송회선을 이용하여 연결하여 정보를 교환하도록 구성한 통신망

> 주요 기능) 정보교환 / 자원공유 / 원격제어
> 

**네트워크 기본 구조**

`호스트 + 네트워크 장비 + 통신매체 + 메세지`

**네트워크 기본 분류**

1. **근거리 네트워크 LAN**
    
    여러 대의 컴퓨터와 정보 기기가 전용 통신 회선을 이용하여 연결되어 있는 네트워크
    
    한정된 지역 (소규모 통신망)
    

1. **광역 네트워크 WAN**
    
    둘 이상의 LAN이 넓은 지역에 걸쳐 연결된 네트워크
    
    라우터를 이용하여 연결 (지역 / 국가 / 대륙)
